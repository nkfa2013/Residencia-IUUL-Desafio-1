HTML (HyperText Markup Language)

Os sites oficiais do HTML são: www.w3.org/html/ e whatwg.org.


Definição:

HTML, ou HyperText Markup Language, é uma linguagem de marcação que define a estrutura e o conteúdo das páginas da web. É o componente mais básico da web e a espinha dorsal de qualquer site.

O HTML é composto por tags, que são códigos divididos por símbolos de marcação “<>”. Cada tag representa um segmento da estrutura do site, como cabeçalho, menu, título, parágrafo, entre outros.

O HTML não é uma linguagem de programação, pois não consegue criar funcionalidades dinâmicas. No entanto, é possível usar outras tecnologias, como CSS e JavaScript, para descrever a aparência e o comportamento de uma página da web.

O HTML é usado para:
° Definir a estrutura e o posicionamento dos elementos
° Dividir uma página em blocos visuais
° Combinar texto, imagens e vídeos
° Organizar e formatar documentos
° Incorporar hiperlinks


Prós e Contras:

Prós:
° É compatível com todos os principais navegadores web
° É fácil de aprender, com uma sintaxe simples
° É escalável, podendo ser usado em projetos de todos os tamanhos e complexidades
° É gratuito e de código-aberto
° Integra-se facilmente com linguagens de back-end como PHP e Node.js
° Os padrões oficiais da internet são mantidos pelo Consórcio World Wide Web (W3C)

Contras:
° Só cria páginas estáticas e simples
° É necessário escrever muito código para criar uma página da web simples
° É preciso acompanhar as tags obsoletas e não usá-las


Grandes projetos que utilizam a tecnologia:

Google (www.google.com)
° O mecanismo de busca mais utilizado do mundo, o Google utiliza HTML para exibir suas páginas de resultados e outros serviços, como o Google Maps e Google Drive.

Wikipedia (www.wikipedia.org)
° A enciclopédia colaborativa online, que permite que usuários de todo o mundo editem e criem artigos em diversas línguas. A estrutura das páginas é baseada em HTML.

YouTube (www.youtube.com)
° A plataforma de compartilhamento de vídeos mais popular, onde usuários podem assistir, enviar e comentar vídeos. As páginas do YouTube são construídas usando HTML.

Facebook (www.facebook.com)
° A rede social mais famosa do mundo, onde usuários podem interagir, compartilhar conteúdo e conectar-se com outras pessoas. O Facebook utiliza HTML para renderizar seu conteúdo para os navegadores.

Amazon (www.amazon.com)
° A maior plataforma de e-commerce do mundo, onde os usuários podem comprar e vender produtos. O layout das páginas e a exibição de produtos são feitos em HTML.

X ou Twitter (www.x.com)
° A rede social focada em microblogging, onde os usuários compartilham mensagens curtas chamadas "tweets". As páginas do Twitter são construídas em HTML para exibir os posts e interações dos usuários.
