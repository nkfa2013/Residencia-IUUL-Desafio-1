JavaScript

Os sites oficiais do JavaScript são: Mozilla Developer Network (MDN) e ECMAScript.


Definição:

JavaScript é uma linguagem de programação que permite criar páginas interativas na internet. É uma das tecnologias mais importantes da World Wide Web e é utilizada para: Atualizar feeds de páginas de mídia social, Exibir animações e mapas interativos, Criar carrossel de imagens, Mudar dinamicamente as cores dos elementos de uma página, Controlar multimídias.

O JavaScript é uma linguagem de alto nível, o que significa que não usa codificações tradicionais de máquina, como “0” e “1”. É uma linguagem versátil e amigável para iniciantes.

Algumas características do JavaScript são:
° É uma linguagem de tipagem fraca, ou seja, o JavaScript reconhece os tipos de variáveis de forma dinâmica.
° É uma linguagem multi-paradigma e dinâmica, suportando estilos imperativos e declarativos.
° É uma linguagem orientada a objetos, baseada em protótipos e em first-class functions (funções de primeira classe).

O JavaScript é uma das tecnologias padrão da web, juntamente com HTML e CSS.


Prós e Contras:

Prós:
° O JavaScript permite criar interfaces de usuário dinâmicas e interativas. Com ele, você pode responder a eventos do usuário, manipular o DOM (Document Object Model) e atualizar o conteúdo da página em tempo real, melhorando a experiência do usuário.
° JavaScript é uma das linguagens de programação mais populares do mundo, com uma grande comunidade de desenvolvedores. Isso resulta em uma vasta quantidade de recursos, bibliotecas, frameworks (como React, Angular e Vue.js) e suporte para resolver problemas comuns, além de uma ampla oferta de documentação e tutoriais.
° Além de ser executado em navegadores, o JavaScript pode ser utilizado no lado do servidor com ambientes como Node.js. Isso permite que os desenvolvedores usem a mesma linguagem tanto para o front-end quanto para o back-end, facilitando o desenvolvimento de aplicações completas.

Contras:
° O JavaScript é suscetível a várias vulnerabilidades de segurança, como ataques XSS (Cross-Site Scripting) e CSRF (Cross-Site Request Forgery). O código JavaScript executado no navegador pode acessar informações sensíveis do usuário, o que requer cuidados especiais em relação à segurança.
° Embora a maioria dos navegadores modernos ofereça bom suporte para JavaScript, ainda existem algumas discrepâncias na implementação de recursos e APIs. Isso pode levar a problemas de compatibilidade que exigem testes e soluções alternativas para garantir uma experiência consistente.
° Apesar de ser uma linguagem bastante rápida, o JavaScript pode ser mais lento em certas operações complexas ou de processamento intensivo em comparação com linguagens compiladas, como C ou C++. Isso pode ser um problema em aplicações que exigem alto desempenho.


Grandes projetos que utilizam a tecnologia:

Google (www.google.com)
° O mecanismo de busca mais utilizado do mundo, o Google utiliza HTML para exibir suas páginas de resultados e outros serviços, como o Google Maps e Google Drive.

Wikipedia (www.wikipedia.org)
° A enciclopédia colaborativa online, que permite que usuários de todo o mundo editem e criem artigos em diversas línguas. A estrutura das páginas é baseada em HTML.

YouTube (www.youtube.com)
° A plataforma de compartilhamento de vídeos mais popular, onde usuários podem assistir, enviar e comentar vídeos. As páginas do YouTube são construídas usando HTML.

Facebook (www.facebook.com)
° A rede social mais famosa do mundo, onde usuários podem interagir, compartilhar conteúdo e conectar-se com outras pessoas. O Facebook utiliza HTML para renderizar seu conteúdo para os navegadores.

Amazon (www.amazon.com)
° A maior plataforma de e-commerce do mundo, onde os usuários podem comprar e vender produtos. O layout das páginas e a exibição de produtos são feitos em HTML.

X ou Twitter (www.x.com)
° A rede social focada em microblogging, onde os usuários compartilham mensagens curtas chamadas "tweets". As páginas do Twitter são construídas em HTML para exibir os posts e interações dos usuários.
