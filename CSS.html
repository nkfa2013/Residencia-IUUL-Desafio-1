CSS (Cascading Style Sheets)

O site oficial do CSS é: Cascading Style Sheets home page, no W3C


Definição:

CSS é uma linguagem de marcação que permite a personalização da aparência de elementos HTML, como cores, fontes, tamanhos de texto, espaçamentos e bordas. A sigla CSS significa Cascading Style Sheets, que em português significa Folha de Estilo em Cascatas.

O CSS é baseado em regras, que são definidas para aplicar grupos de estilo a elementos ou grupos de elementos em uma página web. Por exemplo, é possível definir que o título principal de uma página seja mostrado em vermelho e em tamanho grande.

O CSS é amplamente utilizado com HTML ou XHTML. Algumas das vantagens do CSS são:
° Facilidade de personalização
° Diminuição da repetição de conteúdo na estrutura do código
° Manutenção de um padrão de formatação em diferentes navegadores
° Controle de layout em apenas uma folha
° Criação de formatações com designs mais responsivos

A anatomia de um comando CSS é composta por um seletor, uma propriedade e um valor. O seletor identifica o elemento HTML ao qual o comando será aplicado, a propriedade especifica o atributo a ser alterado e o valor define o novo valor do atributo.


Prós e Contras:

Prós:
° O CSS permite que o conteúdo e o estilo de um site sejam separados, facilitando a manutenção e a atualização do design sem alterar o HTML. Isso melhora a organização do código e permite que várias páginas compartilhem a mesma folha de estilo.
° O CSS oferece uma ampla variedade de opções de estilização, permitindo a personalização detalhada do layout, fontes, cores e animações. Com o uso de CSS moderno (como Flexbox e Grid), é possível criar layouts responsivos que se adaptam a diferentes tamanhos de tela.
° Como o CSS pode ser carregado uma única vez e aplicado a várias páginas, isso pode reduzir o tempo de carregamento em comparação com estilos embutidos em cada página. Além disso, as folhas de estilo podem ser armazenadas em cache pelo navegador, melhorando ainda mais a performance do site.

Contras:
° Em projetos grandes e complexos, o gerenciamento de CSS pode se tornar complicado. Estilos podem se sobrepor, e pode ser difícil rastrear quais regras estão aplicadas a quais elementos, especialmente se não houver uma estrutura organizada.
° Embora a maioria dos navegadores modernos suporte CSS de forma consistente, existem algumas diferenças na forma como diferentes navegadores interpretam certas propriedades. Isso pode levar a inconsistências de aparência entre os navegadores, exigindo testes adicionais e, às vezes, soluções alternativas.
° O CSS tem limitações quando se trata de implementar estilos dinâmicos que dependem de interações de usuário mais complexas. Embora o CSS possa lidar com alguns efeitos de hover e transições, para interações mais sofisticadas ou condicionais, é necessário recorrer a JavaScript, o que pode aumentar a complexidade do código.


Grandes projetos que utilizam a tecnologia:

Google (www.google.com)
° O mecanismo de busca mais utilizado do mundo, o Google utiliza HTML para exibir suas páginas de resultados e outros serviços, como o Google Maps e Google Drive.

Wikipedia (www.wikipedia.org)
° A enciclopédia colaborativa online, que permite que usuários de todo o mundo editem e criem artigos em diversas línguas. A estrutura das páginas é baseada em HTML.

YouTube (www.youtube.com)
° A plataforma de compartilhamento de vídeos mais popular, onde usuários podem assistir, enviar e comentar vídeos. As páginas do YouTube são construídas usando HTML.

Facebook (www.facebook.com)
° A rede social mais famosa do mundo, onde usuários podem interagir, compartilhar conteúdo e conectar-se com outras pessoas. O Facebook utiliza HTML para renderizar seu conteúdo para os navegadores.

Amazon (www.amazon.com)
° A maior plataforma de e-commerce do mundo, onde os usuários podem comprar e vender produtos. O layout das páginas e a exibição de produtos são feitos em HTML.

X ou Twitter (www.x.com)
° A rede social focada em microblogging, onde os usuários compartilham mensagens curtas chamadas "tweets". As páginas do Twitter são construídas em HTML para exibir os posts e interações dos usuários.
